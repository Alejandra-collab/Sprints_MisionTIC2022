# Sprints_MisionTIC2022

### Este es un proyecto realizado en el desarrollo del diplomado en Desarrollo de Software de la Universidad Tecnológica de Colombia (UTP) en el marco del proyecto MisionTIC 2022 del Ministerio de las TIC de Colombia.
Se busca realizar algunos sprints de práctica para el desarrollo de una página web con HTML, CSS y JavaScript. En el proceso se utilizó la metodología SCRUM para el desarrollo de proyectos.
